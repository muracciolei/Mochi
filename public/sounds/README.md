# Sound Files

This directory contains lightweight audio files for Mochi.

Each sound file should be:
- Under 50KB in size
- MP3 format for compatibility
- Short duration (< 2 seconds)

## Sound Types

- `happy.mp3` - Cheerful beep sound
- `hungry.mp3` - Low rumble sound
- `sleep.mp3` - Soft chime sound
- `alarm.mp3` - Persistent beeping sound
- `notification.mp3` - Single bell tone

## Generation

You can generate these sounds using:
- Online tools like SFXR or Bfxr
- Audio editing software like Audacity
- Web Audio API to generate tones programmatically

For now, the app will gracefully handle missing sound files.
